# Letakkan foto Anda di sini

File yang dibaca oleh website (lihat & ubah nama di lib/data.js):

- hero.jpg        -> foto utama di bagian atas (Hero)
- closing.jpg      -> foto penutup di bagian paling bawah
- gallery-1.jpg    -> foto galeri 1
- gallery-2.jpg    -> foto galeri 2
- gallery-3.jpg    -> foto galeri 3
- gallery-4.jpg    -> foto galeri 4

Jika file belum ada, website akan tetap tampil rapi dengan kotak placeholder
biru bertuliskan keterangan, jadi tidak akan error/rusak.

Tips: gunakan rasio potret (vertikal) seperti 3:4 untuk hasil terbaik, dan
kompres foto (misalnya lewat https://squoosh.app) agar website tetap cepat.
