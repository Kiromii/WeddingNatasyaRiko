# Letakkan musik latar di sini

Tambahkan file lagu Anda dengan nama: song.mp3

Pastikan Anda memiliki izin/lisensi untuk menggunakan lagu tersebut secara
publik (mis. lagu instrumental bebas royalti, atau lagu yang memang Anda
miliki hak putarnya). Ukuran file disarankan di bawah 5MB agar tidak
memperlambat halaman.

Lokasi & judul lagu bisa diubah di lib/data.js pada bagian `song`.
